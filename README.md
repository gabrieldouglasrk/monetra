# Monetra

O **Monetra** é um aplicativo de gerenciamento financeiro desenvolvido para ajudar usuários a controlar suas finanças de forma simples, prática e organizada.

Com ele é possível:

- 💰 Registrar receitas (entradas) e despesas (saídas);
- 📊 Acompanhar o saldo disponível em tempo real;
- 📂 Organizar movimentações por categorias;
- 📅 Visualizar o histórico financeiro;
- 🎯 Definir metas de economia;
- ⚙️ Criar regras para melhor organização das finanças;
- 📈 Gerar relatórios para análise de receitas, despesas e desempenho financeiro;
- 🔍 Filtrar informações por período ou categoria.

O objetivo do Monetra é oferecer uma solução intuitiva para o controle financeiro pessoal, permitindo que os usuários acompanhem seus gastos, planejem o orçamento e tomem decisões financeiras com mais segurança.

## Funcionalidades

- Cadastro de receitas e despesas;
- Controle de saldo;
- Gerenciamento de categorias;
- Histórico de transações;
- Relatórios financeiros;
- Metas de economia;
- Regras de organização financeira;
- Interface simples e fácil de utilizar.

## Objetivo

Fornecer uma ferramenta eficiente para o gerenciamento financeiro, incentivando uma melhor organização das finanças pessoais por meio de um ambiente moderno, seguro e intuitivo.
